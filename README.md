# 個人儀表板

一個簡潔美觀的個人儀表板網頁，包含即時時鐘、待辦清單和激勵名言。

## 功能特色

- ⏰ **即時時鐘**：顯示當前時間和日期
- ✅ **待辦清單**：可以新增、完成和刪除待辦事項
- 💫 **激勵名言**：每日激勵人心的名言佳句
- 💾 **資料儲存**：待辦事項會自動儲存在瀏覽器中

## 如何使用

1. 直接在瀏覽器中開啟 `index.html` 檔案
2. 或者使用本地伺服器（推薦）：
   - 如果已安裝 Python：在終端機執行 `python -m http.server 8000`，然後在瀏覽器開啟 `http://localhost:8000`
   - 如果已安裝 Node.js：可以使用 `npx http-server` 或 `npx serve`

## 檔案說明

- `index.html` - 主頁面結構
- `style.css` - 樣式設計
- `script.js` - 功能邏輯

## 使用提示

- 在待辦事項輸入框中輸入內容後，按 Enter 鍵或點擊「新增」按鈕即可新增
- 勾選核取方塊可以標記待辦事項為已完成
- 點擊「刪除」按鈕可以移除待辦事項
- 所有待辦事項會自動儲存，重新整理頁面後仍會保留
